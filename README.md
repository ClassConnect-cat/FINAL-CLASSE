# Backend 
si vols que sigui un port especific, en la carpeta de backend hem de crear l'arxiu .env i possar el seguent codi:
 - Codi de .env
```
PORT=3000
```

## - Fer "npm install" en el backend

## - Fer "npm install" en el frontend

## Fer "docker-compose up" a la carpeta principal

## - Fer "npm run dev" en el frontend

Sortira el seguent en la terminal, ves al link que surt per veure la web:

```
  VITE v5.4.21  ready in 611 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help

```
